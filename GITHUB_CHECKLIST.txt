First-Time Pushers

Step-by-step for first upload:

1. Verify Git is installed:
   git --version

2. In project directory, initialize repo:
   cd c:\Users\MHR\Desktop\talk_to_your_data
   git init

3. Check .gitignore is in place:
   ls -la | grep gitignore

4. Stage all files (excludes .env, __pycache__, .venv, *.log):
   git add .

5. Verify .env is NOT staged:
   git status | grep .env
   (Should show nothing)

6. Create initial commit:
   git commit -m "Initial commit: Talk to Your Data AI Analyst

   - NL-to-SQL Flask API with 17 endpoints
   - RBAC, JWT, rate limiting, SQL injection prevention
   - Fallback mode for offline/quota-safe operation
   - Enhanced UI with caching, charts, export
   - Full documentation and edge-case test coverage"

7. Create GitHub repo (EMPTY, no README/license/gitignore):
   - Go to github.com/new
   - Name: talk-to-your-data
   - Description: Production-ready NL-to-SQL analytics with LLM or fallback mode
   - Make it Public
   - Do NOT initialize with README, .gitignore, or license

8. Add remote:
   git remote add origin https://github.com/your-username/talk-to-your-data.git
   git branch -M main
   git push -u origin main

9. Add GitHub topics (Settings > About):
   - python, flask, ai, nlp, sql, postgresql, llm, natural-language-processing, text-to-sql

10. Add LICENSE (GitHub will show "Add license"):
    - Choose MIT or Apache 2.0

11. Enable Issues & Discussions (Settings):
    - Allow Issues: Yes
    - Allow Discussions: Yes (optional)
